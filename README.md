tinybasic
=========

A tiny BASIC parser, interpreter and compiler. In progress...

##Usage

###REPL
``` python
python tinybasic.py
```
###Interpret file
``` python
python tinybasic.py [path]
```

###Parse file
``` python
python tinybasic.py [path] -p
```
