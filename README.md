tinybasic
=========

A tiny BASIC parser, interpreter and compiler.

##Usage

###REPL
``` python
python tinybasic.py
```

###Interpret file
```
python tinybasic.py [path]
```

###Parse file
```
python tinybasic.py [path] [-p, --parse]
```

###Compile file to C ... at your own risk
```
python tinybasic.py [path] [-c, --compile]
```
